CS99DBSQL requires [MSSMS](https://www.microsoft.com/en-us/sql-server/sql-server-downloads) to run.

You can eather run it by the prepare.bat in sqls folder or with sql runner of MSSMS

```sh
$ cd sqls
$ ./prepare.bat
```
